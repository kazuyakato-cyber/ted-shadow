🚀 セットアップガイド（初回のみ）

1
GitHubアカウントを作成
github.com でSign up（無料）
2
新しいリポジトリを作成
「New repository」→ 名前を ted-shadow に設定 → 「Create repository」
3
このHTMLファイルをアップロード
「Add file」→「Upload files」→ この shadowing-ted.html をドラッグ → 「Commit changes」
4
GitHub Pagesを有効化
Settings → Pages → Branch: main を選択 → Save
数分後にURLが発行されます
https://あなたのID.github.io/ted-shadow/shadowing-ted.html
5
そのURLをブラウザで開く
スマホのSafari/ChromeでURLを開けばYouTubeも録音も全機能使えます 🎉
使い方
▶

英文・日本語訳・発音のポイントを動画直下で確認。画面切り替え不要。
🎙

自分の声を録音して聴き直し。発音・リズムを客観的にチェック。
